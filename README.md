# PDF Table Extractor (Streamlit)

Convert scanned PDF tables to Excel using OCR (Tesseract).

## Features
- Upload PDF files
- OCR-based extraction for scanned PDFs
- Custom column headers
- Excel export with formatting
- CSV export
- Adjustable OCR settings

## Tech Stack
- Streamlit
- pytesseract
- pdf2image
- pandas
- openpyxl
